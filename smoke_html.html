Methods


Statistical Analysis

A bivariate random-effects meta-analysis was performed using the hierarchical model described by Reitsma et al. (2005). This approach jointly models sensitivity and specificity while accounting for the correlation between these measures and heterogeneity across studies.

Model specification:

• Primary model: Bivariate generalized linear mixed model (GLMM)

• Estimation: Maximum likelihood (ML) via Fisher scoring

• Link function: Logit transformation

• Random effects: Study-specific intercepts for sensitivity and specificity

• Correlation: Estimated between-study correlation (ρ)

Summary measures:

• Pooled sensitivity and specificity with 95% confidence intervals

• Summary receiver operating characteristic (SROC) curve

• Area under the curve (AUC) using the bivariate model

• Diagnostic odds ratio (DOR) with 95% CI

• Positive and negative likelihood ratios (LR+, LR-)

Heterogeneity assessment:

• Cochran's Q statistic with associated p-value

• I² statistic with 95% confidence interval

• Between-study variance (τ²)

• Prediction intervals for expected accuracy in new settings

Publication bias:

• Deeks' funnel plot asymmetry test

• Harbord test for small-study effects (DTA-specific)

Quality assessment:

• QUADAS-2 tool for methodological quality

All analyses were performed using DTA Meta-Analysis Pro v4.9.2.
