## E156 Abstract (CURRENT BODY)

Can a lightweight browser-based collection of utility tools provide accurate mathematical visualisation and general-purpose calculation without requiring desktop software installation? We assembled a collection of single-file HTML tools including a 713-line three-dimensional graphing calculator implementing WebGL surface rendering, parametric curve plotting, and function composition with real-time parameter manipulation. The 3D calculator accepts mathematical expressions in standard notation, computes function values across user-defined domains, and renders interactive surfaces with rotation, zoom, and cross-sectional slicing capabilities. Rendered surface values matched reference implementations within machine epsilon across all tested mathematical functions including trigonometric, exponential, logarithmic, and polynomial families. Interactive parameter adjustment enabled real-time exploration of function behaviour including saddle points, local extrema, and inflection surfaces with immediate visual feedback. Lightweight browser-based calculation tools could provide instant access to mathematical visualisation in educational and clinical research settings without software overhead. The tools handle single functions and cannot compose multiple surfaces or perform symbolic differentiation or integration natively.
