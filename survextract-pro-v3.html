SurvExtract Pro v3 - Help

Quick Start

  1. Load a survival curve image
  2. Select the curve type (KM, CI, Hazard)
  3. Calibrate axes by clicking 3 points
  4. Use Auto-Trace or manually digitize
  5. Reconstruct IPD and validate
  6. Export for meta-analysis

New in v3

  • Guyot Validation: Overlay reconstructed curve on original
  • Bootstrap CI: Uncertainty estimation for IPD
  • Cox Regression: Proper HR from partial likelihood
  • Cross-Validation: RMSE, correlation metrics
  • Dark/Light Theme: Toggle with 🌓 button
  • Interactive Tutorial: Step-by-step guidance

Keyboard Shortcuts

  • 1-5: Switch modes (Calibrate/Digitize/Censor/CI/Pan)
  • Ctrl+Z: Undo last action
  • Space: Toggle pan mode
  • +/-: Zoom in/out
  • 0: Fit to screen
  • T: Toggle theme
